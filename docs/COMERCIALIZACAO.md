# Comercialização do NexaGest

## Build

Use:

```powershell
npm install --foreground-scripts
npm run dist:win
```

O instalador é gerado como:

```txt
NexaGest-Setup.exe
```

## Atualizações

A partir da versão 9.0.3, o NexaGest usa `electron-updater` e GitHub Releases com:

```txt
latest.yml
NexaGest-Setup.exe
NexaGest-Setup.exe.blockmap
```

## Próximas etapas comerciais

- Licenciamento online.
- Área do cliente.
- Assinatura digital do instalador.
- Site de download.
