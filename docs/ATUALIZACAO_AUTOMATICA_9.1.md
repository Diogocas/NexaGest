# Atualização automática — NexaGest 9.1

O NexaGest agora usa o fluxo profissional do `electron-updater` com GitHub Releases.

## Por que às vezes não aparece atualização?

A atualização só aparece quando a versão da última Release é maior que a versão instalada.

Exemplo:

- Instalado: 9.1.0
- Última Release: 9.1.0
- Resultado: Sistema atualizado

Para testar atualização real:

- Instale uma versão antiga, por exemplo 9.0.3.
- Publique uma Release `v9.1.0`.
- Clique em **Verificar atualização agora**.

## Arquivos da Release

Anexe somente:

```txt
NexaGest-Setup.exe
latest.yml
NexaGest-Setup.exe.blockmap
```
