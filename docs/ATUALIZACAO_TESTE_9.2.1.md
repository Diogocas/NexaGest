# NexaGest 9.2.1 — Teste de Atualização

Esta versão foi criada para testar o fluxo real de atualização automática.

Fluxo recomendado:

1. Instale a versão 9.2.0 no computador.
2. Gere o instalador da 9.2.1.
3. Publique uma Release no GitHub com a tag `v9.2.1`.
4. Anexe somente:
   - `NexaGest-Setup.exe`
   - `latest.yml`
   - `NexaGest-Setup.exe.blockmap`
5. Abra o NexaGest 9.2.0 instalado.
6. Aguarde a verificação automática ou use Comercialização > Verificar atualização agora.

Observação: o repositório/Releases precisam estar públicos para o app instalado acessar sem token.
