# NexaGest 13.1.0 - Trial automático e bloqueio de licença

## O que foi adicionado

- Primeiro uso cria automaticamente um teste gratuito de 7 dias.
- O teste usa todos os recursos do sistema durante o período.
- Faltando 3 dias ou menos, o sistema avisa ao iniciar.
- Se o sistema estiver aberto quando a licença vencer, a checagem automática bloqueia o uso.
- Após expirar, somente a tela de ativação continua disponível.
- Os dados do cliente ficam preservados.
- Uma licença por período ou permanente gerada no License Manager libera o sistema novamente.

## Observação

A licença de teste automática é criada uma única vez por instalação/base local. Apagar a chave manualmente não gera outro teste se a instalação já tiver criado um trial antes.
