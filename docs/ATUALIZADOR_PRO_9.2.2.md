# NexaGest 9.2.2 — Atualizador PRO

Esta versão substitui a janela padrão do Electron por uma janela integrada ao visual do NexaGest.

## Fluxo

1. O app instalado verifica o GitHub Releases.
2. Quando encontra uma versão nova, abre um modal do NexaGest.
3. O usuário clica em Atualizar agora.
4. O download mostra progresso, tamanho e velocidade.
5. Ao terminar, o app oferece Reiniciar e instalar.

## Release

Anexe somente:

- NexaGest-Setup.exe
- latest.yml
- NexaGest-Setup.exe.blockmap

O repositório/Release precisa estar público para o app instalado consultar sem token.
