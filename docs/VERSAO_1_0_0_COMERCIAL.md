# NexaGest 1.0.0 — Primeira versão comercial

Esta versão marca o fechamento do primeiro ciclo comercial do NexaGest.

## Ajustes finais

- Versão do sistema ajustada para `1.0.0`.
- Instalador mantido com nome padrão `NexaGest-Setup.exe`.
- Comprovantes de venda em HTML/PDF/impressão agora exibem o aviso:
  - Este documento é um comprovante de venda.
  - Não possui valor fiscal.
- Modelo padrão de comprovante via WhatsApp também inclui aviso de documento sem valor fiscal.

## Observação

O comprovante de venda não substitui NFC-e, NF-e, SAT, ECF ou qualquer documento fiscal exigido pela legislação.
