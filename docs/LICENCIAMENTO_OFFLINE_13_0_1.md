# Histórico - Licenciamento offline 13.0.1

Esta versão simplifica o licenciamento para dois modelos:

- **Por período**: 7, 15, 30, 90, 180, 365 dias ou quantidade personalizada. A contagem começa na primeira ativação no computador do cliente.
- **Permanente**: não expira.

A geração de licenças fica somente em `tools/NexaGest-License-Manager-DEV.html`. A tela do cliente apenas ativa a licença e mostra tipo, data de ativação e validade.

A partir da v13.0.2, o conceito de planos foi removido.
