# Licenciamento NexaGest

## Modelo inicial
- Licença local no formato `NEXA-XXXX-XXXX-XXXX`.
- Modo demonstração com chave `DEMO`.
- Status salvo nas configurações locais.

## Modelo recomendado para venda
- Conta por e-mail/senha.
- Assinatura mensal/anual.
- Token offline com validade de 7 a 30 dias.
- Renovação automática quando houver internet.
