# NexaGest — Primeiro uso

1. Configure os dados da empresa em **Configurações > Empresa**.
2. Cadastre usuários e permissões.
3. Configure backup local ou pasta sincronizada na nuvem.
4. Cadastre produtos principais, custo, venda e estoque mínimo.
5. Abra o caixa, faça uma venda de teste e feche o caixa.
6. Gere um relatório de vendas para conferir os dados.

## Login padrão de desenvolvimento
Usuário: `admin`  
Senha: `admin`

Altere a senha antes de entregar para cliente.
