# NexaGest 9.3.2 — Atualizador Premium

Esta versão melhora a experiência de atualização automática:

- limpa HTML/README bruto das novidades exibidas no modal;
- padroniza textos internos para 9.3.2;
- mantém atualização via GitHub Releases com `latest.yml`;
- melhora diagnóstico visual na tela Comercialização, licença e atualização.

## Release

Na Release do GitHub, anexe somente:

- `NexaGest-Setup.exe`
- `latest.yml`
- `NexaGest-Setup.exe.blockmap`

O corpo da Release pode ter texto simples em markdown, por exemplo:

```
- Atualizador Premium com novidades limpas
- Melhorias de diagnóstico
- Correções internas
```
