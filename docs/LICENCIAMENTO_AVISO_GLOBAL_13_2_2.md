# NexaGest 13.2.2 - Aviso global de licença

Adicionado aviso global ao lado do relógio quando a licença estiver próxima do vencimento.

- Mais de 3 dias: nenhum aviso global.
- 3 dias ou menos: aviso amarelo/laranja/vermelho.
- Licença permanente: sem aviso.
- Licença expirada: mantém a tela de bloqueio.
