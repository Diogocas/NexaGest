# NexaGest 13.2.3 - Dados automáticos da licença

- O License Manager DEV continua exigindo cliente e e-mail para controle comercial.
- Cliente e e-mail ficam embutidos na licença gerada.
- No NexaGest instalado, o usuário cola apenas a licença e clica em Ativar.
- O sistema preenche automaticamente cliente, e-mail, tipo e validade.
- O histórico local de licenças geradas fica salvo no License Manager DEV.
