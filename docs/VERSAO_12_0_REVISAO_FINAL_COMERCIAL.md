# NexaGest 12.0.0 — Revisão Final Comercial

Esta versão não adiciona dependência de aplicativo mobile, API online ou serviços pagos.
O foco é preparar o NexaGest Desktop para testes comerciais, demonstração e primeira venda.

## O que foi revisado

- Versão do projeto atualizada para 12.0.0.
- PUBLICAR.bat atualizado para publicar a versão 12.0.0.
- Tela de Diagnóstico recebeu checklist de prontidão comercial.
- Exportação de checklist comercial em TXT.
- Revisão de textos de licença, publicação e atualização.
- Mantido o foco em funcionamento local/offline.

## Checklist recomendado antes de vender

1. Configurar dados reais da empresa.
2. Criar usuários separados para administrador, caixa e estoque.
3. Cadastrar produtos de demonstração ou produtos reais.
4. Testar abertura e fechamento de caixa.
5. Testar venda em dinheiro, cartão e PIX manual.
6. Testar relatório, etiqueta, orçamento e financeiro.
7. Gerar backup antes de publicar atualização.
8. Gerar instalador e testar em outro computador.
9. Publicar release no GitHub somente depois dos testes.

## Observação

A versão Mobile foi pausada para evitar custo e complexidade com servidor/API online neste momento.
O NexaGest segue preparado para evoluir para Cloud/API/Mobile futuramente, quando o projeto já estiver gerando retorno.
