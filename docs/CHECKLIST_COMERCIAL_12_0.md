# Checklist Comercial NexaGest

Use este checklist antes de entregar uma instalação para cliente.

## Empresa
- [ ] Nome da empresa configurado
- [ ] Telefone/WhatsApp configurado
- [ ] Cidade/endereço revisados
- [ ] Logo conferido

## Operação
- [ ] Produtos cadastrados
- [ ] Estoque mínimo revisado
- [ ] Clientes cadastrados
- [ ] Fornecedores cadastrados
- [ ] Compras testadas
- [ ] Orçamentos testados
- [ ] Etiquetas testadas

## Caixa e vendas
- [ ] Abrir caixa testado
- [ ] Venda testada
- [ ] PIX manual testado
- [ ] Comprovante/recibo testado
- [ ] Fechar caixa testado

## Segurança
- [ ] Usuário administrador protegido
- [ ] Senha padrão alterada
- [ ] Usuário operador criado
- [ ] Permissões revisadas

## Publicação
- [ ] Backup feito
- [ ] Instalador gerado
- [ ] Teste em outro computador
- [ ] PUBLICAR.bat revisado
- [ ] GitHub Release preparado
