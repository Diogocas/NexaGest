# NexaGest 13.2.4 - License Manager PRO

Melhorias no NexaGest License Manager DEV:

- Histórico local com busca por cliente, e-mail ou licença.
- Renovar licença gerando nova chave para o mesmo cliente.
- Adicionar dias gerando nova chave por período.
- Converter cliente para licença permanente.
- Copiar licença diretamente do histórico.
- Excluir registros apenas do histórico local.
- Exportar histórico em JSON ou CSV.
- Importar backup JSON do histórico.

Observação: como o licenciamento é offline, renovar ou adicionar dias gera uma nova chave. O cliente precisa colar essa nova chave no NexaGest para atualizar a validade.
